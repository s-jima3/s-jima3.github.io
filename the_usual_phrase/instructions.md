# The Usual

いつもの」フレーズを簡単に送信！

### "The Usual"のダウンロードに感謝いたします。

## はじめに（概要）
アプリ名 : The Usual  
目的 : 定型文送信アプリ  

## 簡単な説明
いつもの」フレーズを簡単に送信！
日常のための定型文共有ツールです。

Lineや+メッセージなどのチャット、メールやメモなど、いつものアプリへ定型文(フレーズ)を送り込みます。  

作成した定型文はタイトルをつけて管理できるので、選択時の間違いを減らし、色を付けることで老眼の方でも利用しやすくしました。  

いつでもクイック設定パネルからのタップで、いつものアプリに簡単に定型文を送り込めますので、メッセージアプリなどでの活用に便利です。  

文字列を受け取れないアプリともクリップボード経由で利用可能です。  

## 利用環境（動作環境）
Android 12以上

## 操作手順（画面ごとの詳細）

### メイン画面

#### ハンバーガーメニュー(アプリ名横)
[メニュー](#メニュー)を表示し、追加や寄付、他を操作します。  

#### [ + ] ボタン
タップで新しい定型文を作成します。  

#### 送信先のアプリ
[ アプリ選択 ] ボタン タップで定型文を送るお気に入りのいつものアプリを選びます。  
※クイック設定パネルで送り込むアプリにもなります。  
※選択画面 左下の [ 共有で処理 ] はいつものアプリでは無く、デフォルトの OS 共有画面で送り込むアプリを毎回選べます。  
※選択画面 右下の [ クリップボード ] はいつものアプリでは無く、OS のクリップボードへ送り込まれます。  

#### [ 定型文 ] ボタン
各ボタン以外の部分をタップすることで、定型文がお気に入りのアプリに送り込まれます。  
※長押しで、アプリが設定されていても共有メニューを開くことができます。  

##### チェックボックス
チェックされたものはクイック設定パネルで表示されます。  
※クイック設定パネルの表示件数は、初期３つ+寄付回数×２件です。  

##### [ アプリ送信(紙飛行機) ] ボタン
設定したお気に入りのアプリ(共有・クリップボード)へ定型文が送り込まれます。  

##### ラベル
定型文の名前(ラベル)です。  
分かりやすい名前をつけてください。  
分かりやすいように色もつきます。  

##### 本文
本文の頭数行だけ表示されます。  

##### [ コピー ] ボタン
定型文がアプリでは無く、クリップボードに送り込まれます。  

##### [ 編集 ] ボタン
設定した定型文を編集できます。  

##### [ 上下 ] ボタン
定型文ボタンの位置を上下に移動できます。  

### 新しい定型文 / 定型文を編集 画面

#### ラベル(一覧で表示される)
分かりやすい定型文の名前をつけてください。  

#### 本文
送り込む定型文を入力します。  

#### ラベルカラー
定型文およびラベルの色を選択するか、カラーコードを直接編集してください。  

#### [ 追加する / 変更を保存する ] ボタン
追加・編集した内容で保存します。  
※保存後はチェックが外れるのでご注意ください。  

#### [ キャンセル ] ボタン
追加・編集をキャンセルします。  

#### [ 削除 ] ボタン
定型文を削除します。  
※削除の確認画面が表示されます。  

### メニュー

#### + 新しい定型文を追加
メイン画面の[ + ] ボタンと同じ(新しい定型文を作成します)。  

#### ♥ 寄付で枠を増やす
缶コーヒー代を寄付してもらうことで、開発中に喉を潤せます。  
クイック設定パネルで呼び出せる、定型文を(１缶コーヒーで+定型文２つ)増やせるようにしました。  

#### 背景色を反転

##### 背景
白/黒 背景色が反転します。
